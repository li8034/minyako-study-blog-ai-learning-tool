# 上传与隐私说明

## 不应上传到公共仓库的内容

- `blog/content/settings.local.json`
- 个人 API Key、访问令牌与私有接口地址
- 个人导入的文章目录 `blog/content/pages/<page-id>/`
- 本机缓存、临时测试文件、外部实验目录
- 嵌套仓库目录，例如当前已忽略的 `acac/`

## 当前上传版的保留内容

- 博客前端页面与样式
- 本地 Python 服务端
- 默认分类与公共配置模板
- 空内容目录与导入说明
- 产品名与基础使用文档

## 建议的发布流程

1. 先检查 `git status --short`
2. 确认没有出现 `settings.local.json`、个人页面目录或缓存目录
3. 确认新增文件是否属于产品代码或公共文档
4. 再执行提交与推送

## 当前仓库已特别处理的风险项

- 根目录下的 `acac/` 为本地额外克隆的仓库，内部包含独立 `.git` 历史与作者信息，已通过根目录 `.gitignore` 排除
- 上传版默认不追踪 `blog/content/pages/` 下的具体文章页面，只保留说明文件
