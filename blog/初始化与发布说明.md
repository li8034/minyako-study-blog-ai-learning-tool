# 初始化与发布说明

## 1. 无个人信息初始化版本

项目已经提供一份不包含个人 API Key 的初始化模板：

```text
starter/clean-content/
```

如果你想把当前项目恢复成干净版本，可以：

1. 备份当前 `content/`
2. 用 `starter/clean-content/` 中的文件替换 `content/` 内对应文件
3. 按需保留你自己的 `pages/` 内容

## 2. 本地私有配置

仓库默认配置放在：

```text
content/settings.json
```

本地私有配置建议放在：

```text
content/settings.local.json
```

你可以直接复制：

```text
content/settings.local.example.json
```

然后改名为：

```text
content/settings.local.json
```

再填写你自己的 AI 配置。
