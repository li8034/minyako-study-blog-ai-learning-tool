# 初始化与发布说明

## 1. 无个人信息初始化版本

项目已经提供一份不包含个人 API Key 的初始化模板：

```text
starter/clean-content/
```

如果你想把当前项目恢复成干净版本，可以：

1. 备份当前 `content/`
2. 用 `starter/clean-content/` 中的文件替换 `content/` 内对应文件
3. 按需保留你自己的 `pages/` 内容

## 2. 本地私有配置

仓库默认配置放在：

```text
content/settings.json
```

本地私有配置建议放在：

```text
content/settings.local.json
```

你可以直接复制：

```text
content/settings.local.example.json
```

然后改名为：

```text
content/settings.local.json
```

再填写你自己的 AI 配置。

## 3. 建议发布到 GitHub 的内容

建议提交：

- 前端页面与样式
- Python 服务端
- `starter/clean-content/`
- 示例文章或公共学习资料
- 项目说明文档

建议不要提交：

- `content/settings.local.json`
- 任何带个人 API Key 的配置
- 本机缓存文件

## 4. 建议的仓库展示名称

推荐展示名称：

**Minyako的学习日志：一款 AI 辅助学习工具**

如果 GitHub 仓库名需要英文 slug，可使用：

```text
minyako-study-log-ai-learning-tool
```
